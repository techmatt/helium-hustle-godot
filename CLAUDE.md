## Project Overview

Helium Hustle is an idle game built with Godot 4.6 (C#/.NET via dotnet). The project is in early development — no scenes or scripts exist yet.

## Engine & Configuration

- **Godot version**: 4.6
- **Language**: C# (dotnet assembly configured as "AIComplete")
- **Rendering**: GL Compatibility (D3D12 on Windows)
- **Physics**: Jolt Physics (3D)
- **Project root**: `godot/` subdirectory

## Running the Project

Open the project in Godot 4.6 by pointing the editor at the `godot/` directory (open `godot/project.godot`). There is no CLI build command — use the Godot editor to run, export, or test.

## Repository Structure

```
godot/          # Godot project root (open this in the editor)
  project.godot # Project config
  .godot/       # Editor cache — do not commit (gitignored)
```

## Notes

- The `project.godot` has `config/name="AIComplete"` which is a placeholder — the intended project name is "Helium Hustle".
- The `.godot/` directory is auto-generated by the editor and should not be manually edited or committed.
- Since this uses C# (dotnet), scripts will be `.cs` files, not `.gd` GDScript files.
